🧊 立体几何 (调换 B↔D · B₁↔D₁)
棱长为 1 的正方体 · 上下底面调换 · B↔D, B₁↔D₁
✅ D 选项 · 外接球表面积
三棱锥 A–A₁PD 的外接球的表面积为 41π / 16
📐 推导证明 (新坐标下)
调换后坐标：A(0,1,0), A₁(0,0,0), D(1,1,0), P(0, 0.5, 1)
设球心 O(x,y,z)，由 OA=OA₁=OD=OP 得：
x² + (y–1)² + z² = R²  ……①
x² + y² + z² = R² ⇒ y = 1/2
(x–1)² + (y–1)² + z² = R² ⇒ x = 1/2
x² + (y–0.5)² + (z–1)² = R² ⇒ z = 3/8
球心 O(1/2, 1/2, 3/8)
R² = (1/2)² + (1/2)² + (3/8)² = 1/4 + 1/4 + 9/64 = 41/64
S = 4πR² = 4π × 41/64 = 41π / 16 ✅
